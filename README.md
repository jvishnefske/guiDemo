# guiDemo

A lightweight Python desktop application that renders HTML5 canvas content inside a native Qt window using PySide2 and QWebEngineView. Ideal for building hybrid desktop interfaces that combine native menus with web-based visualization.

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```

## Features

- Native Qt5 window with embedded HTML5 canvas
- File menu with reload (Ctrl+R) and exit (Ctrl+Q) shortcuts
- Fullscreen canvas support
- Cross-platform (Linux, macOS, Windows)
